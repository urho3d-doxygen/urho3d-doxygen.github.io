btCollisionShape * Urho3D::CollisionShape::UpdateDerivedShape ( int  shapeType,
const Vector3 &  newWorldScale 
)
protectedvirtual

Called when instantiating a collision shape that is not one of ShapeType (default no-op).

Useful for custom shape types that subclass CollisionShape and use a non-standard underlying btCollisionShape. UpdateDerivedShape can then be overridden to create the required btCollisionShape subclass.
